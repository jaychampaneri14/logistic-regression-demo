# Logistic Regression Comprehensive Demo

A complete tutorial on logistic regression covering binary/multiclass classification, decision boundaries, and interpretability.

## Topics Covered
1. **Sigmoid function** visualization
2. **Binary classification** with probabilistic decision boundary
3. **Multiclass** (OvR vs multinomial)
4. **Regularization strength** (C parameter) effect
5. **Odds ratios** for feature interpretability

## Setup

```bash
pip install -r requirements.txt
python main.py
```

## Output
- `sigmoid_function.png` — sigmoid curve visualization
- `binary_classification.png` — decision boundary + ROC curve
- `multiclass.png` — 4-class boundary
- `regularization_effect.png` — 5 values of C compared
- `odds_ratios.png` — feature importance via odds ratios
